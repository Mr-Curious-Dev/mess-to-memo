# Roadmap

Mess-to-Memo already supports the production path from Telegram or Toon export to a local `notion-ready/` package.

Planned additions:

- direct Notion publication helpers once page targets are configured;
- more fixture exports and expected `notion-ready/` outputs;
- stricter validation for generated manifests and safety reports;
- dataset templates in JSON or YAML;
- optional packaging for plugin-style distribution.
