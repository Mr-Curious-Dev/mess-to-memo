# Companion Project: Chat-to-Memory Review Kit

Demonstrates the review-first workflow from the article "From Telegram Chaos to Project Memory: How We Built Mess-to-Memo at Maximus Studio".

## What This Shows

- how to explain the pipeline in one glance
- how to review extracted project memory before publication
- how to shape a Notion-ready package for human approval

## Contents

| File | Purpose |
|------|---------|
| `examples/chat-to-memory-workflow.mmd` | Mermaid workflow diagram of the full pipeline |
| `templates/project-memory-review-template.md` | Review checklist for extracted knowledge |
| `examples/notion-ready-package-outline.md` | Example layout for a publish-ready package |

## How To Use It

1. Open the Mermaid file to understand the pipeline.
2. Use the review template during documentation QA.
3. Compare your local output against the sample package outline.

## Article Reference

- **Article**: From Telegram Chaos to Project Memory: How We Built Mess-to-Memo at Maximus Studio
- **Sections**: "What Mess-to-Memo actually does", "Built for review, not blind publishing"
